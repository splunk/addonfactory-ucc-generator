Getting Started =============== To get started with JavaScript Lint, download the appropriate package and extract its contents. * JavaScript Lint 0.3.0 for Windows ([jsl-0.3.0-win32.zip](http://www.javascriptlint.com/download/jsl-0.3.0-win32.zip)) * JavaScript Lint 0.3.0 for Mac OS X Intel ([jsl-0.3.0-mac.tar.gz](http://www.javascriptlint.com/download/jsl-0.3.0-mac.tar.gz)) * JavaScript Lint 0.3.0 Source ([jsl-0.3.0-src.tar.gz](http://www.javascriptlint.com/download/jsl-0.3.0-src.tar.gz)) Developers can also check out from the Subversion trunk: > svn co https://javascriptlint.svn.sourceforge.net/svnroot/javascriptlint/trunk jsl The [documentation](/docs/) explains the different ways that you can run JavaScript Lint. JavaScript Lint continues to be improved. If you encounter problems or have suggestions or revisions, please visit the project [forum](http://sourceforge.net/forum/?group_id=168518) and [tracker](http://sourceforge.net/tracker/?group_id=168518).
