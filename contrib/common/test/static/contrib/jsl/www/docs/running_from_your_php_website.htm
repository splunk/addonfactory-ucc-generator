Running from Your PHP Website ============================= The JavaScript Lint source package (available from the [download page](/download.htm)) includes a `_jsl_online.php` file that allows PHP websites to integrate with JavaScript Lint. This script is used to power [the online lint](/online_lint.php). The following example usage is included in the PHP script: > require_once("_jsl_online.php"); $engine = new JSLEngine('.priv/jsl', '.priv/jsl.server.conf'); $result = $engine->Lint($code); if ($result === true) OutputLintHTML($engine); else echo '' . htmlentities($result) . '';
